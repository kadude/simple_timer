# simple_timer
Simple timer written in rust... Just to learn rust.

This is just a test application to learn rust. Learn by doing, I guess.

## Usage
### Countdown to time
`simpleTimer 10:30`

### Countdown number of seconds
`simpleTimer 3600`
`simpleTimer $((60*60))`

### Countdown hours, minutes and seconds
`simpleTimer 1h`
`simpleTimer 1h 30m`
`simpleTimer 1h 30m 5s`
`simpleTimer 30m 5s`
